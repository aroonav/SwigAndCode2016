A binary string of length N is a string of N characters, where each character is either 0 or 1.
There exists a function named fun1() which maintains a list named binStrings, consisting of all 2^N N-digit(as N places can be filled in 2 ways each) binary strings.
For each string X in binStrings, fun1() executes a function fun2(X), defined as follows (psuedocode):
   bool fun2(x):
     //note the zero indexing
     n = length(x)
     counter = 0
     FOR i = 1 to n-1
         IF x[i] == '0' and x[i - 1] == '0'
             INCREMENT counter
         END IF
     ENDFOR
     return (counter == 1)
Now, fun1() stores an element X in a new list, binTrues if fun2(X) returns True for that X.
In other-words, binTrues consists of all the elements X in binStrings such that fun2(X) returns True.
Calculate the length of binTrues for each N. I/P 2 3 O/P 1 2
Explanation:
When N=2, the list binStrings is equal to ['00', '01', '10', '11'].
Calling fun2() on each element in binStrings, the function only returns true when X is equal to 00. When X = 01, 10, 11, the counter in fun2() is equal to 0.
Therefore, binTrues is equal to ['00'], and has a length of 1.
When N=3, the list binStrings is equal to ['000', '001', '010', '100', '011', '101', '110', '111'].
Calling fun2() on each element in binStrings, the function only returns true when X is equal to 001 or100.
Therefore, binTrues is equal to ['001', '100'], and has a length of 2.
